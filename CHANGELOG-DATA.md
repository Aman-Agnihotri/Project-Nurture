# Data Changelog

under `project_nurture/public/demo/` (data tier T2). Each entry lists the
artifact, the exact generator command used to produce it, and its sha256
digest so any change to the committed bytes is traceable and reproducible.
See also `project_nurture/public/demo/data-manifest.json`, which tracks the
same provenance information in machine-readable form.
