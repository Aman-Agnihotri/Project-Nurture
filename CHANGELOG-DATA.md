# Data Changelog

This changelog records every change to the committed public data artifacts
under `project_nurture/public/demo/` (data tier T2). Each entry lists the
artifact, the exact generator command used to produce it, and its sha256
digest so any change to the committed bytes is traceable and reproducible.
See also `project_nurture/public/demo/data-manifest.json`, which tracks the
same provenance information in machine-readable form.

## 2026-07-21

### district_risk.json

- Generated by:
  `SOURCE_DATE_EPOCH=1784592000 python python_backend/analytics/risk_methodology.py`
- Reads only the committed T2 `district_indicators.json` artifact. It ranks
  stunting, underweight, and wasting across included demo districts, then uses
  the documented 0.45 / 0.35 / 0.20 percentile composite and six renormalized
  sensitivity scenarios.
- All source district values and resulting scores remain documented demo
  placeholders, not authoritative NFHS estimates.
- sha256: `27a7f5bd3019d6654c7534caa595a3d00c5447e211473d062b6651fc2e20da70`

## 2026-07-11

### district_indicators.json

- Generated by:
  `SOURCE_DATE_EPOCH=1783728000 python python_backend/demo_data_builder.py --seed 42`
- Contains one record for each committed fact-sheet demo district and state
  rollups calculated as **unweighted, demo-tier** means. It is derived only
  from `python_backend/data/nfhs5_district_factsheet.csv`.
- All values remain documented placeholders, not real NFHS estimates.
- sha256: `17d812c76b502a9e9b05563783fffbc2b990f7ee9ae1ffb464f1e93b83d45f20`
